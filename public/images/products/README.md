Drop product photos here and they will automatically appear on the storefront,
replacing the SVG illustration. Use exactly these filenames (all lowercase, JPG):

Recommended size: **~800×600 px**, JPG or PNG, max ~200 KB each.

To add more overrides, edit `src/components/ProductImage.jsx` →
`photoOverrides` object.

For wide **banner photos** (lifestyle / context shots), use
`/public/images/banners/` instead.
